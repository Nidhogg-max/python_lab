# 🧪 Python 个人实验室

我的 Python 学习与实验空间，主要包含教材练习和知识笔记两个部分。

## 📁 目录结构

```
.
├── ython_homework       # 教材配套练习（个人代码实现）
└── python_notebook      # 学习笔记（指令、基础与进阶知识）
```

### `python_homework`
> 注：目录名为 `ython_homework`（原样保留），实际为 Python 课后习题及教材示例的本地练习代码。

- 按章节/主题组织
- 每个 `.py` 文件对应一个独立练习
- 包含运行示例和简单注释

### `python_notebook`
使用 Jupyter Notebook 记录的学习笔记，涵盖：

- **常用指令**：Python 内置函数、模块导入、文件操作等
- **基础知识**：数据类型、流程控制、函数、类与对象
- **进阶知识**：装饰器、生成器、上下文管理器、常用标准库（如 `itertools`、`collections`）、异常处理、测试入门等

## 🚀 如何使用

1. 克隆本仓库
   ```bash
   git clone https://github.com/你的用户名/你的仓库名.git
   ```
2. 运行练习代码
   ```bash
   cd ython_homework
   python 某个练习.py
   ```
3. 查看笔记
   - 本地启动 Jupyter Notebook：`jupyter notebook`
   - 或直接在 GitHub 上预览 `python_notebook` 中的 `.ipynb` 文件（如渲染失败可借助 [nbviewer](https://nbviewer.org/)）
