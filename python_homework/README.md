## 📖 章节内容概览

### 第一章 Python 基础
- **作业 1**：逻辑运算符 `or` 的返回值（短路求值）
- **作业 2**：`int()` 向下取整
- **作业 3**：字节串 (`bytes`) 与字符串 (`str`) 的区别与编码转换
- **作业 4**：`eval()` 函数的作用与安全用法
- **作业 5**：`zip` 与 `dict` 创建字典
- **作业 6**：使用 `random` 生成列表并计算最大、最小、平均值
- **作业 7**：`print` 函数中 `sep` 与 `end` 参数的区别
- **作业 8**：列表排序多种方式：`sorted` 的 `key`、`reverse`、`itemgetter` 及 `reversed` 等

### 第二章 函数式编程初步
- **作业 1**：`map` 函数的行为——以最短的可迭代对象长度为准
- **作业 2**：`filter` 函数过滤字符串（`isalnum`），对比列表推导式、`lambda` 等多种写法

### 第三章 列表推导式与内置函数
- **作业 1**：`all`、`any`、`max` 与 `key` 参数的使用
- **作业 2**：列表推导式展平二维列表
- **作业 3**：普通循环与 `itertools.chain` 展平列表
- **作业 4**：筛选列表中 0～10 之间的元素
- **作业 5**：字典统计成绩的最高分、最低分、平均分，并找出最高分的人

#### 第三章选做
- **选做 1**：用列表推导式和 NumPy 两种方法找出列表中满足条件的元素索引
- **选做 2**：用 `map` 和 NumPy 提取二维列表的第一列

### 第四章 控制结构与基础编程
- **作业 1**：统计随机生成的 1000 个字符的频率（利用字典）
- **作业 2**：`enumerate` 遍历列表并格式化输出
- **作业 3**：三元表达式（`value_if_true if condition else value_if_false`）
- **作业 4**：计算 1～100 中所有能被 3 整除的数的和

#### 第四章选做
- **选做 1**：输入三个数，升序输出并给出最值（两种实现：`sorted` 与手动变量交换）
- **选做 2**：分数到等级的转换函数（含异常处理）
- **选做 3**：PyTorch 术语字典查找，使用 `get` 方法处理不存在的键

### 第五章 循环与生成器
- **作业 1**：打印九九乘法表
- **作业 1 修改**：输出 1～100 内能被 7 整除但不能被 5 整除的数
- **作业 2**：找出 100 以内的最大素数，并改写为打印所有素数
- **作业 3**：使用生成器 (`yield`) 实现斐波那契数列，输出前 10 项

### 第六章 函数定义
- **作业 1**：判断素数的函数，结合 `main()` 交互输入
- **作业 2**：定义 `stats(*numbers)` 函数，计算最大值与总和

#### 第六章选做
- **选做 1**：找出所有的三位水仙花数（153、370、371、407）
- **选做 2**：理解 `global` 关键字的作用——函数内修改全局变量

### 第七章 字符串操作
- **作业 1**：字符串方法 `find`、`rfind`、`index`、`count` 的区别与异常处理
- **作业 2**：将字符串中偶数索引位置的字符转换为大写
- **作业 3**：凯撒加密实现，支持自定义偏移量
- **作业 4**：`jieba` 分词库的 `cut`（生成器）与 `lcut`（列表）的使用
